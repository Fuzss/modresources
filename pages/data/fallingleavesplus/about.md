**Falling Leaves Plus** enhances Minecraft's falling leaf particles with detailed, tree-specific visuals that make forests feel more alive and distinctive. Instead of every tree shedding the same simple particle, each leaf type can produce its own unique collection of leaves, branches, acorns, cones, and other natural debris. 🍃

The mod brings personality to every forest biome. Oak trees release acorns, spruce trees can drop cones, and many leaf types feature multiple particle variations for a richer and more natural appearance. Snow-covered leaves release snowflakes, adding extra atmosphere to winter landscapes. ❄️

Beyond the built-in effects, this mod is fully resource pack driven. Custom particle textures and behaviors can be added for modded trees and resource packs, making it easy to extend the system far beyond vanilla Minecraft. 🎨
