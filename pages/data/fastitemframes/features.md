* 🧱 **Block-Based Item Frames:** Item frames are implemented as blocks instead of entities for greatly improved performance.  
* 🚀 **Less Lag, More FPS:** Rendering and updates are much faster, giving FPS boosts in areas with many frames.  
* 💥 **Better Interactions:** Includes proper breaking particles and full compatibility with commands like `/setblock`.  
* ⚡ **Instant Reactions:** Frames immediately respond when their supporting block is broken or moved by pistons.  
* ✅ **Multiple Frames Per Block:** Still supports multiple item frames occupying the same block space.  
* 🎨 **Millions of Colors:** Dye item frames just like leather armor to unlock 5,713,438 unique colors.  
* 🫥 **Invisible Frames:** Shift + right-click with an empty hand to hide the frame itself while displaying the item larger than usual.  
* 🍯 **Waxed Item Frames:** Use honeycomb to lock item frames and prevent accidental rotation, modification, or visibility changes.  
* 🏷️ **Optional Name Tag Hiding:** Prevent custom item names from appearing while looking at displayed items.  
* 🔄 **World Conversion Options:** Optionally convert existing vanilla item frame entities into item frame blocks upon loading chunks.
