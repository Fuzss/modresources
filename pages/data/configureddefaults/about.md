**Configured Defaults** is a utility mod for modpacks that automatically copies missing files and folders into the game or server directory from a default location provided through the `configureddefaults` folder. 📦

Its primary purpose is helping modpack creators distribute default configuration files. This is especially useful when adding new mods in an update, ensuring players receive the intended configuration instead of having the mod generate its own defaults on first launch. ⚙️

The mod supports any file or folder within the game directory, making it useful for far more than just configuration files. 📁

Additionally, `options.txt` receives special handling. Rather than replacing an existing file, missing entries can be merged from the provided default version, allowing new settings to be added while preserving a player's existing preferences. ✨
