This mod ships with several optional data packs that add new enchantment cross compatibilities and enchantment item compatibilities. Each pack focuses on a specific item or enchantment group and can be enabled independently.

The packs can be enabled or disabled from the `Data Packs` screen when creating a world, or via the `/datapack` command in existing worlds and multiplayer servers.

Detailed documentation, a complete reference of all included data packs, and information on the additional data pack mechanics provided by this mod can be found in the `README.md` on GitHub.
